\f0\fs24 \cf0 # Project Notes\
\
## Objective\
Build a property intelligence dataset and dashboard to identify and prioritize high-value waterfront properties for targeted outreach.\
\
## Key Steps\
- Combined active listings and recently sold properties\
- Engineered a dock_score based on property features\
- Created segmentation and ranking logic\
- Built A/B/C priority tiers\
- Added targeting strategy recommendations\
- Developed a dashboard for decision support\
\
## Challenges\
- Limited access to structured property data\
- Manual data collection was time-intensive\
- Ownership data was inconsistent\
\
## Future Improvements\
- Add geographic segmentation\
- Expand to additional lakes\
- Convert to a web-based dashboard}
